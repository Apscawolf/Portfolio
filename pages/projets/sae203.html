SAE 2.03

MateZone - Application de Chat Collaborative

Développement d'une application de messagerie instantanée en Java avec architecture client-serveur et déploiement Docker

📅 Semestre 2 👥 Travail en équipe ☕ Java Swing 🐳 Docker

Aperçu du Projet

MateZone est une application de chat collaborative développée en Java Swing permettant la communication en temps réel entre utilisateurs. Le projet implémente une architecture client-serveur avec déploiement Docker pour une solution de messagerie complète.

🎯 Objectif

Développer une application de messagerie instantanée complète avec interface graphique moderne, serveur multi-clients et déploiement Docker professionnel.

👥 Équipe

Projet en équipe avec répartition des rôles : développement client, serveur, interface graphique et DevOps. Développement collaboratif avec Git et repository partagé.

🏆 Résultats

Application fonctionnelle avec interface moderne, serveur stable gérant plusieurs clients simultanés et déploiement automatisé avec Docker.

Réflexion et Apprentissages

💡 Ce que j'ai appris

Ce projet m'a fait découvrir les complexités du développement réseau et de l'architecture client-serveur. La gestion des threads et de la synchronisation était particulièrement challengeante, mais très formatrice pour comprendre les enjeux de la programmation concurrente.

L'aspect Docker était nouveau pour moi et m'a ouvert les yeux sur l'importance du déploiement et de la portabilité des applications. Configurer un environnement multi-services dans un conteneur m'a appris les bases du DevOps et de l'administration système.

Le travail en équipe sur un projet technique de cette ampleur m'a appris l'importance de la communication et de la documentation. Coordonner le développement de modules interdépendants demande une organisation rigoureuse et une bonne maîtrise des outils de versionning comme Git.

Au niveau technique, maîtriser Java Swing pour créer des interfaces graphiques complexes et réactives m'a donné une solide base en programmation orientée événements. La gestion des fichiers pour la persistance des données m'a également fait comprendre l'importance de bien structurer ses données.
